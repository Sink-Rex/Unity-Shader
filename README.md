## 渲染流水线
### 渲染流程的三个阶段：应用阶段、几何阶段、光栅化阶段

应用阶段：开发者主导的阶段，输出渲染所需的结合信息即渲染图元。
几何阶段：与渲染图元打交道，重要任务是输出屏幕空间的二位顶点坐标、每个顶点对应的深度值、着色等相关信息。
光栅化阶段：决定每个渲染图元中的那些像素应该绘制在屏幕上。

几何和光栅化阶段在GPU上处理。

### CPU和GPU之间的通信
- (1)把数据加载到显存：从硬盘加载到RAM再到GPU
- (2)设置渲染状态：渲染的状态
- (3)调用Draw Call：CPU告诉GPU渲染那个
